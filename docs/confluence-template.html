Overview

Publish-confluence is a powerful solution for embedding interactive web applications, dashboards, and widgets directly into Confluence. With a single command, you can push your latest build to a Confluence page, complete with all required assets and custom layout. Whether you're publishing interactive dashboards, implementing documentation-as-code, automating API documentation, or creating multi-page documentation structures with nested hierarchies—this tool eliminates manual uploads, brittle macros, and tedious copy-paste workflows.

Designed for rapid iteration, publish-confluence lets you update your app, rebuild, and immediately publish changes to Confluence—making it ideal for fast feedback, demos, and continuous delivery. The tool's flexible template system and smart asset management enable you to deliver rich, production-quality experiences inside Confluence, while keeping your workflow simple and repeatable.

Key Features

  • Instant App Publishing: Deploy JavaScript/TypeScript applications to Confluence pages in seconds.
  • Seamless Asset Management: Automatically uploads and attaches all build assets (JS, CSS, images, fonts).
  • Customizable Templates: Use Handlebars for full control over page and macro layout.
  • Macro Helper Library: Easily compose Confluence-native macros (panels, layouts, code blocks, etc.).
  • Effortless Configuration: Simple JSON config and environment variables.
  • Supports Modern Workflows: Designed for Vite, Webpack, Rollup, etc.
  • Robust CLI Experience: Includes commands for scaffolding, fetching, and prompt generation.
  • Nested Page Publishing: Manage hierarchical documentation structures.
  • CI/CD Integration: Automate publishing through GitHub Actions, GitLab CI, Jenkins, or Azure DevOps.

Core Concepts

Templates: Control the structure of your Confluence page (`confluence-template.html`) and the embedded HTML macro (`macro-template.html`) using Handlebars.

Asset Management: The tool automatically finds your build output (JS, CSS, images), uploads them as attachments, and injects the correct links into your macro template via `{{{scripts}}}` and `{{{styles}}}`.

Macro Helpers: Use built-in Handlebars helpers (like \{{#confluence-panel}} or \{{confluence-toc}}) within your page template to generate rich Confluence content alongside your embedded application.

Hierarchical Content: Create and maintain complex documentation structures with nested child pages, all managed through a single configuration file.

Classic Use Cases

  • Interactive Dashboards: Embed real-time data visualizations (React, Vue, Preact, etc.).
  • Training Materials: Create interactive documentation with live code examples.
  • Internal Tools: Deploy team collaboration tools (planning poker, simple forms) directly in your workspace.
  • Documentation Widgets: Enhance documentation with small, interactive components.
